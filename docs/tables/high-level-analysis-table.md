# High-Level Analysis Table

`corashighlevelanalysistable` is the canonical table environment. Use `\corashighlevelanalysisrow` for rows.

## Example

```latex
\begin{corashighlevelanalysistable}[caption={Documenting the high-level analysis}]
  \corashighlevelanalysisrow
    {Hacker}
    {Gets access to a privileged account}
    {Weak authentication controls}
\end{corashighlevelanalysistable}
```
